Prerequisites (2 minutes)

What You Need

  • ✓ Two CentOS 8 / RHEL 8 VMs
  • ✓ SSH access to both VMs
  • ✓ 5GB+ disk space per VM
  • ✓ Control machine with Python 3.9+
  • ✓ Ansible 2.10+ (for Ansible approach)

IP Addresses

Note the IP addresses of your VMs:

  • VM1 (Primary): 192.168.56.101 (update with your IP)
  • VM2 (Secondary): 192.168.56.102 (update with your IP)

Setup SSH (2 minutes)

1. Generate SSH Key

ssh-keygen -t rsa -b 4096 -f ~/.ssh/id_rsa -N ""

2. Copy to Both VMs

ssh-copy-id -i ~/.ssh/id_rsa.pub root@192.168.56.101
ssh-copy-id -i ~/.ssh/id_rsa.pub root@192.168.56.102

3. Test SSH Access

ssh -i ~/.ssh/id_rsa root@192.168.56.101
exit
ssh -i ~/.ssh/id_rsa root@192.168.56.102
exit

Pro Tips

  • 💡 Use strong passwords (uppercase, lowercase, numbers, special chars)
  • 💡 Test SSH connectivity before deployment
  • 💡 Use dry-run mode with Ansible: -C flag
  • 💡 Monitor logs during first deployment
  • 💡 Keep SSH keys secure
  • 💡 Backup VMs before first deployment
